# ImóvelAI — AI Real Estate Platform

An intelligent real estate platform powered by Claude AI, featuring:

- 🤖 Sofia — 24h AI assistant for client support
- 🔍 Natural language property search
- 👥 Automatic lead capture and qualification
- 📅 Visit scheduling
- 💰 Financing simulator
- 🎯 AI-powered lead scoring
- 📄 AI contract generation
- 📊 Real-time dashboard

## Stack

- **Frontend:** HTML, CSS, JavaScript
- **AI:** Claude API (Anthropic) via Netlify Functions
- **Database:** Supabase
- **Auth:** Google OAuth via Supabase
- **Deploy:** Netlify
